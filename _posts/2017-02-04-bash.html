选项: -e: 支持反斜线控制的字符转换

普通输出，不带-e

echo hello

修饰字符的输出，带-e的输出

echo -e 'h\te\tl\nl\to'
控制字符 作用
\a 输出警告音
\b 退格键，向左删除键
\n 换行符
\r 回车符
\t 制表符，也就是Tab键
\v 垂直制表符
\0nnn 八进制ASCII
\xhh 十六进制ASCII

bash脚本执行方式

#!/bin/bash
#the first program
echo -e "\e[1;32m 天上掉下个林妹妹 \e[0m"
  • #30m = 黑色
  • #31m = 红色
  • #32m = 绿色
  • #33m = 黄色
  • #34m = 蓝色
  • #35m = 洋色
  • #36m = 青色
  • #37m = 白色
