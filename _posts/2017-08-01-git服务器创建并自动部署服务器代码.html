一，创建git仓库

1，方法一

sudo git init --bare xxx.git
// 成功创建 返回如下
Initialized empty Git repository in /www/dongshushu/git/xxx.git/

1，方法二

mkdir xxx.git
cd xxx.git
git init --bare
// 返回如下
Initialized empty Git repository in /www/dongshushu/git/xxx.git/
